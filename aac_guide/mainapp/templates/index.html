AAC FRIEND는 사용자와 그 가족, 언어재활사, 도구 개발자, 연구자 등 모두에게 열린 공간입니다.

웹사이트 제작

이 페이지는 언어학특강 발표용으로 제작되었습니다.

Front-end

 사용자의 눈에 보이는 디자인이나 인터페이스 영역

사용자의 입력을 서버로 보내주는 역할

Back-end

사용자의 요청을 서버에서 받는 역할로 기능 구현이나 데이터베이스 관리를 포함

인스타그램은 'django'로 구축한
대표적인 웹사이트

Web Server

 다른 사용자들이 인터넷에서 이 웹사이트로 접속할 수 있게 웹 서버를 구축하는 과정이 필요함

AWS는 웹 서버 구축을 도와주는
아마존의 클라우드 컴퓨팅 서비스

research topic

보완대체의사소통(AAC)에 대한 인식 제고 - AAC 정보 접근성 향상과 웹/앱 개발 활성화를 중심으로 -
AAC와 프로그래밍
정보 접근성
웹/앱 개발

AAC에 대한 정보를 찾기가 힘듦

 AAC를 알고 있는 사람들이 많지 않을 뿐더러 사용자나 언어 재활사, 특수교사가 AAC를 활용할 수 있는 방법을 찾기 힘들다.

기존 AAC 도구들이 가진 문제점들

 기존 AAC 도구들은 가격이 비싸고, 여러 장애 유형이나 개인의 특성을 고려하지 않는 경우가 많았다. 또한 개인이 선택할 수 있는 도구의 종류가 다양하지 않다.
